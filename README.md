# 🌟 Jojo-OS v3.0

> "A aventura bizarra da produtividade."

Jojo-OS é um ambiente de desktop personalizado desenvolvido em Python, focado em uma interface bizarra, rápida e cheia de estilo. Transforme o seu fluxo de trabalho em uma jornada épica com ferramentas integradas e uma estética única.

## ✨ Funcionalidades

- 🎭 **Interface:** Área de trabalho interativa baseada em janelas (Tkinter).
- 🤖 **AI:** Assistente virtual inteligente integrado com Google Gemini.
- 🧮 **Calculater:** Calculadora de precisão com design dourado.
- 🎮 **Arcade Mode:** Acesso direto a jogos integrados (Pygame).
- 📂 **Inventory Manager:** Gestão de arquivos e diretórios do sistema(nao adicionado ainda).

## 🚀 Como Executar

1. **Pré-requisitos:**
   - Python 3.10 ou superior.
   - Uma chave de API do Google Gemini (colocada no arquivo `.env`).

2. **Instalação:**
   ```bash
   pip install requirements.txt
